Accelerometers and Gyroscopes

This week we will begin working on our self-balancing robot! The first thing we will need to do is learn how to connect and test our Motion Processor Unit (MPU) which contains Accelerometers and Gyroscopes so that our bot can know its orientation.


We will need the MPU6050 library to use our MPU:

MPU6050.zip

We will also need this I2C library:

I2Cdev.zip

Here is our first test code for the MPU:

lcd_mpu_test.zip

Self-Balancing Robot

This is it, the ultimate challenge! Can we build a robot that will balance all on its own?

The robot requires the PID library. Download it and select Sketch > Include Library > Add .ZIP Library

PIDLibrary.zip

Next you'll download the code for the robot here:

self_balancing_bot_pid.zip

To adapt this code to your bot, you will need to fine-tune the PID parameters.
